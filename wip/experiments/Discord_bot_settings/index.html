Upload mission

Upload a .pbo to the Arma server's mpmissions/ folder. Filename must end .pbo and follow the Name.MapName.pbo pattern (e.g. OpIronShield.Altis.pbo) so the server knows which terrain to load. Restart so the server indexes it, then switch in-game with the #missions admin chat command.

max 50 MB
leave blank to use the uploaded file's name
required for the server to see the new mission in its admin list

Installed missions

Files in the server's /mpmissions folder. Sandbox stays the auto-boot default — “Set as boot default” only changes which mission loads on next server start. Admins normally switch missions live via #missions.

Modlist

The server's /modlist.html is what players drag into the Arma launcher to match the server. Either push whatever's currently captured by the modlist Discord channel, or upload a launcher export from your machine directly (any filename — auto-renamed to modlist.html on the server).
